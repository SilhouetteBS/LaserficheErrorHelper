# Product Aliases

The public product list uses canonical product names. Search should still work when users use common aliases.

| Alias | Canonical Product |
| --- | --- |
| Web Access | Web Client |
| Web Scanning | Web Client |
| Repository Server | Laserfiche Server/Repository Server |
| Laserfiche Server | Laserfiche Server/Repository Server |
| LFDS | Directory Server |
| Directory Service | Directory Server |
| Desktop Client | Windows Client/Desktop Client |
| Windows Client | Windows Client/Desktop Client |
| LF Workflow | Workflow |
| LF Forms | Forms |
| Forms Server | Forms |
| Federated Search Service | Federated Search |
| Webtools | Webtools Agent |

When adding new entries, use only the canonical product names from `src/data/catalogMetadata.js`.
